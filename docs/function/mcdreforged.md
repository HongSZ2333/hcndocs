# 添加MCDReforged  
MCDR教程: 

```
[MCDR] [TaskExecutor/INFO]: MCDR 指令帮助信息列表
[MCDR] !!apr: 在文件变更时自动重载插件
[MCDR] !!bot: 管理地毯模组假人
[MCDR] !!calc: 查看计算插件使用帮助
[MCDR] !!cexample: Hello world from command
[MCDR] !!day: 查看服务器运行天数
[MCDR] !!example: Hello world
[MCDR] !!help: 显示指令帮助信息
[MCDR] !!here: 广播坐标并高亮玩家
[MCDR] !!here: 查询玩家的坐标
[MCDR] !!info: 获取服务器信息
[MCDR] !!joinMOTD: 显示欢迎消息
[MCDR] !!leader: leader_reforged.plugin.help_message
[MCDR] !!loc: 路标管理
[MCDR] !!MCDR: MCDR 控制指令
[MCDR] !!msr: MirrorServerReforged 帮助
[MCDR] !!po: 传送/收寄副手物品
[MCDR] !!qb: 备份/回档，5槽位
[MCDR] !!stats: 查询统计信息并构建计分板
[MCDR] !!task: 工程任务进度管理
[MCDR] !!tps help: TPS检测帮助
[MCDR] !!vris: 查询玩家的坐标
[MCDR] !!whereis: 查询玩家的坐标
[MCDR] =<expression>: 等同于 !!calc <expression>
[MCDR] ==<count>: 等同于 !!calc item <count>
[MCDR] @ <someone>: @某人，@ all可作用于所有玩家。使用两个@@小心被打
[MCDR] [GM] !Clear 注册(注册后方可使用!c / !s)
[MCDR] [GM] !c 旁观
[MCDR] [GM] !s 生存
```
